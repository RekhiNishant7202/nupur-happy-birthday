# 🎂 Interactive Birthday Surprise Website

A beautiful, magical birthday website with 10 interactive pages!

## 📁 Project Structure

```
birthday-surprise/
├── index.html          # Main HTML file
├── css/
│   └── style.css       # Styling and animations
├── js/
│   └── script.js       # Interactive functionality
├── images/             # Place your photos here
└── music/              # Place your music here
```

## 🚀 How to Use

### 1. Add Music
Download a "Happy Birthday" MP3 file and place it in the `music/` folder named `happy-birthday.mp3`

### 2. Add Photos
Replace the photo placeholders in `index.html` (page 6) with your actual photos:
- Replace `<div class="photo-placeholder">Photo 1</div>` with your image tag:
  `<img src="images/photo1.jpg" alt="Memory 1" style="width:100%; border-radius:5px;">`

### 3. Customize Content
- Page 6: Update quotes, memories, and wishes
- Page 7: Update the Marathi poem
- Page 8: Customize the apology letter
- Page 9: Customize the final birthday message

### 4. Deploy
You can deploy this website on:
- **GitHub Pages**: Push to GitHub and enable Pages in settings
- **Netlify**: Drag and drop the folder
- **Vercel**: Deploy with Git integration

✅ Marathi poem
✅ Animated letter

✅ QR code generator

## 🎨 Customization Tips

- **Colors**: Modify the `colors` array in `js/script.js`
- **Fonts**: Change Google Font links in `index.html`
- **Messages**: Update the `messages` array in `js/script.js`
- **Music**: Replace `happy-birthday.mp3` with your preferred song
